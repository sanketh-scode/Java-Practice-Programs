# Student Management System (Java)

## Description
This is a console-based Student Management System developed using Java.
The application allows users to manage student records efficiently.

## Features
- Add Student
- View All Students
- Search Student by ID
- Delete Student
- Menu Driven Interface

## Concepts Used
- Object Oriented Programming (OOP)
- Encapsulation
- ArrayList Collection
- CRUD Operations
- Java Scanner Class

## Project Structure
Student.java – Student data model  
StudentManager.java – Business logic  
Main.java – User interface

## How to Run
1. Compile all files:
   javac *.java

2. Run program:
   java Main
